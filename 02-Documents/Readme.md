### It is designed for:

* Journalists

* Attorneys

* Researchers

* Members of the public

The purpose is clarity, neutrality, and traceability. Not argument.

### How to Use This Index

* Each entry represents a single document file

* Descriptions are factual and restrained

* Links point to publicly viewable copies (hosted in this repository)

* Original files are preserved offline and are not altered

## Access to Original Document Files

Original document files are not distributed publicly due to PII of submitter.

Attorneys, journalists, or other qualified reviewers may request access to original document files for verification or review purposes.

Requests should include:
- Name and affiliation
- Purpose of the request
- Intended use of the material

Requests may be directed to:  
**aljusticewatchdog@gmail.com**
